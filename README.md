# Sistema de Gestión de Notas de Alumnos

## Descripción

Este proyecto es una aplicación web interactiva desarrollada con HTML, CSS y JavaScript que permite gestionar las notas de 10 alumnos en tres certámenes.

El sistema permite ingresar datos, calcular promedios y mostrar resultados de forma organizada.

---

## Funcionalidades

### Entrada de datos

* Formulario para ingresar:
  * Nombre del alumno
  * Notas de 3 certámenes
* Validación de datos (campos obligatorios y notas válidas)

### Cálculos

* Promedio individual por alumno
* Promedio del curso por cada certamen
* Promedio general del curso
* Cantidad de alumnos aprobados (≥ 55)
* Cantidad de alumnos reprobados (< 55)

### Resultados

* Tabla con nombre, notas y promedio final de cada alumno
* Lista ordenada de alumnos según su promedio
* Resumen general del curso

---

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (ES6)

---

## Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio.
2. Abrir el archivo `index.html` en un navegador.
3. Ingresar los datos en el formulario.
4. Visualizar los resultados generados.

---

## Validaciones

* Las notas deben estar dentro de un rango válido (por ejemplo, de 0 a 100).
* No se permiten campos vacíos.
* El sistema muestra mensajes en caso de error.
